# Мавка 0.123.0

## Будування

Треба:

- GNU/Linux
- Ціль _0.48.0_
- Clang
- Zig (для компіляції для AArch64/Windows/MacOS)

```shell
sh build.sh debug x86_64-linux

./будування/ікс86_64-лінукс/готове/мавка
```

Будування відбувається лише через GNU/Linux.
Мавку можна збудувати на різні платформи, але працездатність перевіряється насамперед на `x86_64-linux`.

Доступні режими будування:

- `debug`
- `release`

Доступні платформи будування:

- `x86_64-linux`
- `aarch64-linux`
- `x86-windows`
- `x86_64-windows`
- `aarch64-windows`
- `x86_64-macos`
- `aarch64-macos`
