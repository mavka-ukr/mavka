# Мавка 0.123.0

## Будування

Треба:

- GNU/Linux
- Ціль _0.48.0_
- Clang
- Zig (для компіляції для AArch64/Windows/MacOS)

```shell
sh build.sh debug linux-x86_64

./будування/{ВЕРСІЯ}/лінукс-ікс86_64/готове/мавка
```

Будування відбувається лише через GNU/Linux.
Мавку можна збудувати на різні платформи, але працездатність перевіряється насамперед на `linux-x86_64`.

Доступні режими будування:

- `debug`
- `release`

Доступні платформи будування:

- `linux-x86_64`
- `linux-aarch64`
- `windows-x86_64`
- `windows-aarch64`
- `macos-x86_64`
- `macos-aarch64`
